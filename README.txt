Sree Arunachala Infra Group - Bootstrap Website Bundle
-----------------------------------------------------

Includes:
- index.html, about.html, projects.html, gallery.html, contact.html
- assets/bootstrap/bootstrap.min.css (offline placeholder)
- assets/bootstrap/bootstrap.bundle.min.js (offline placeholder)
- assets/css/style.css (minor overrides)
- assets/js/main.js (contact form demo)
- assets/img/*.svg placeholders

How to use:
1. Unzip the package.
2. Open index.html in browser.
3. Replace Bootstrap placeholder CSS/JS with official files from https://getbootstrap.com (if you want full features).
4. Customize text, images, colors.
